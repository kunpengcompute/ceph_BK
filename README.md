# Ceph
## 介绍
Ceph是一个专注于分布式的、弹性可扩展的、高可靠的、性能优异的存储系统平台，可以同时支持块设备、文件系统和对象网关三种类型的存储接口。
详细特性介绍可参考[Ceph官网](https://ceph.com/en/)

## 仓库说明
本分支主要包含在openEuler 20.03操作系统的鲲鹏服务器上容器化/物理机部署并配置UCX的内容

## 支持CPU
华为鲲鹏920处理器

## 支持软件版本
Ceph 14.2.8
UCX 1.14.1
